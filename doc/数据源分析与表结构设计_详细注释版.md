# 数据源分析与TDengine表结构设计（详细注释版）

## 一、数据源数据类型分析

### 1. pytdx（通达信）数据源

#### 1.1 股票数据
| 数据类型 | 说明 | 字段详情 |
|---------|------|---------|
| **实时行情** | 股票实时五档行情 | 代码、名称、最新价、开盘价、最高价、最低价、昨收、成交量、成交额、买一价~买五价、卖一价~卖五价、买一量~买五量、卖一量~卖五量、涨跌额、涨跌幅、换手率、量比、市盈率、市净率、总市值、流通市值 |
| **K线数据** | 日K、周K、月K、1分钟、5分钟、15分钟、30分钟、60分钟 | 时间、开盘、最高、最低、收盘、成交量、成交额、持仓量（期货）、结算价（期货） |
| **Tick数据** | 逐笔成交数据 | 时间、价格、成交量、成交额、买卖方向 |
| **分时数据** | 每分钟分时数据 | 时间、价格、成交量、成交额、均价 |
| **指数行情** | 各类指数实时行情 | 指数代码、名称、最新、开盘、最高、最低、昨收、成交量、成交额、涨跌额、涨跌幅 |
| **股票列表** | 股票基本信息 | 代码、名称、市场、行业、板块、上市日期、退市日期 |
| **财务数据** | 季报、年报财务数据 | 报告期、每股收益、每股净资产、净资产收益率、净利润、营业收入、总资产、负债、股东权益 |
| **分红送配** | 分红送配历史 | 分红年度、每股分红、送股比例、转增比例、除权除息日、股权登记日 |
| **龙虎榜** | 龙虎榜数据 | 日期、股票代码、上榜原因、买入金额、卖出金额、净买入额、买入营业部、卖出营业部 |
| **资金流向** | 资金流向数据 | 日期、主力流入、主力流出、散户流入、散户流出、净流入、净流入率 |
| **板块数据** | 行业板块、概念板块 | 板块名称、板块类型、涨跌幅、成交量、成交额、上涨家数、下跌家数、平盘家数 |
| **成交明细** | 逐笔成交明细 | 时间、价格、成交量、成交额、买卖方向、成交编号 |
| **历史分笔** | 历史分笔数据 | 时间、价格、成交量、成交额 |

#### 1.2 期货数据
| 数据类型 | 说明 | 字段详情 |
|---------|------|---------|
| **实时行情** | 期货实时五档行情 | 合约代码、名称、最新价、开盘价、最高价、最低价、昨收、成交量、持仓量、成交额、买一价~买五价、卖一价~卖五价、买一量~买五量、卖一量~卖五量、结算价、昨结算、涨跌额、涨跌幅、涨停价、跌停价 |
| **K线数据** | 日K、周K、月K、1分钟、5分钟、15分钟、30分钟、60分钟 | 时间、开盘、最高、最低、收盘、成交量、持仓量、成交额、结算价 |
| **Tick数据** | 逐笔成交数据 | 时间、价格、成交量、成交额、持仓量、买卖方向 |
| **合约列表** | 期货合约信息 | 合约代码、名称、交易所、品种、交割月份、上市日期、最后交易日、交割日、最小变动价位、合约乘数、交易单位、交割方式 |
| **持仓明细** | 会员持仓明细 | 日期、合约代码、会员名称、成交量、持仓量、多头持仓、空头持仓、多头增减、空头增减 |

## 二、TDengine表结构完整设计（含详细字段注释）

### 表1: stock_quotes（股票实时行情表）

**说明**: 存储股票实时行情数据，支持十档买卖盘口数据

```sql
CREATE STABLE IF NOT EXISTS stock_quotes (
    -- 数据列（时间序列数据）
    ts TIMESTAMP,                    -- 时间戳，数据采集时间
    open FLOAT,                      -- 开盘价
    high FLOAT,                      -- 最高价
    low FLOAT,                       -- 最低价
    close FLOAT,                     -- 收盘价/最新价
    volume BIGINT,                   -- 成交量（手）
    amount FLOAT,                     -- 成交额（元）
    
    -- 买一到买十价格
    bid_price1 FLOAT,                -- 买一价
    bid_price2 FLOAT,                -- 买二价
    bid_price3 FLOAT,                -- 买三价
    bid_price4 FLOAT,                -- 买四价
    bid_price5 FLOAT,                -- 买五价
    bid_price6 FLOAT,                -- 买六价（Level-2）
    bid_price7 FLOAT,                -- 买七价（Level-2）
    bid_price8 FLOAT,                -- 买八价（Level-2）
    bid_price9 FLOAT,                -- 买九价（Level-2）
    bid_price10 FLOAT,               -- 买十价（Level-2）
    
    -- 卖一到卖十价格
    ask_price1 FLOAT,                -- 卖一价
    ask_price2 FLOAT,                -- 卖二价
    ask_price3 FLOAT,                -- 卖三价
    ask_price4 FLOAT,                -- 卖四价
    ask_price5 FLOAT,                -- 卖五价
    ask_price6 FLOAT,                -- 卖六价（Level-2）
    ask_price7 FLOAT,                -- 卖七价（Level-2）
    ask_price8 FLOAT,                -- 卖八价（Level-2）
    ask_price9 FLOAT,                -- 卖九价（Level-2）
    ask_price10 FLOAT,               -- 卖十价（Level-2）
    
    -- 买一到买十数量
    bid_vol1 BIGINT,                 -- 买一量（手）
    bid_vol2 BIGINT,                 -- 买二量（手）
    bid_vol3 BIGINT,                 -- 买三量（手）
    bid_vol4 BIGINT,                 -- 买四量（手）
    bid_vol5 BIGINT,                 -- 买五量（手）
    bid_vol6 BIGINT,                 -- 买六量（手，Level-2）
    bid_vol7 BIGINT,                 -- 买七量（手，Level-2）
    bid_vol8 BIGINT,                 -- 买八量（手，Level-2）
    bid_vol9 BIGINT,                 -- 买九量（手，Level-2）
    bid_vol10 BIGINT,                -- 买十量（手，Level-2）
    
    -- 卖一到卖十数量
    ask_vol1 BIGINT,                 -- 卖一量（手）
    ask_vol2 BIGINT,                 -- 卖二量（手）
    ask_vol3 BIGINT,                 -- 卖三量（手）
    ask_vol4 BIGINT,                 -- 卖四量（手）
    ask_vol5 BIGINT,                 -- 卖五量（手）
    ask_vol6 BIGINT,                 -- 卖六量（手，Level-2）
    ask_vol7 BIGINT,                 -- 卖七量（手，Level-2）
    ask_vol8 BIGINT,                 -- 卖八量（手，Level-2）
    ask_vol9 BIGINT,                 -- 卖九量（手，Level-2）
    ask_vol10 BIGINT,                -- 卖十量（手，Level-2）
    
    -- 其他指标
    last_close FLOAT,                 -- 昨收价
    limit_up FLOAT,                   -- 涨停价
    limit_down FLOAT,                 -- 跌停价
    open_int BIGINT,                 -- 开盘集合竞价成交量
    pre_close FLOAT,                  -- 前收盘价
    turnover_rate FLOAT,              -- 换手率（%）
    pe_ratio FLOAT,                  -- 市盈率
    pb_ratio FLOAT,                  -- 市净率
    total_cap BIGINT,                -- 总市值（元）
    float_cap BIGINT,                -- 流通市值（元）
    rise_fall FLOAT,                 -- 涨跌额（元）
    rise_fall_rate FLOAT,            -- 涨跌幅（%）
    amplitude FLOAT,                  -- 振幅（%）
    volume_ratio FLOAT                -- 量比
) TAGS (
    -- 标签列（静态数据）
    symbol NCHAR(20),               -- 股票代码（如：000001）
    market NCHAR(10),               -- 市场（SH：上海，SZ：深圳）
    name NCHAR(50),                 -- 股票名称
    industry NCHAR(50),              -- 所属行业
    sector NCHAR(50)               -- 所属板块
);
```

### 表2: stock_bars（股票K线数据表）

**说明**: 存储股票K线数据，支持多种周期（1分钟、5分钟、15分钟、30分钟、60分钟、日K、周K、月K）

```sql
CREATE STABLE IF NOT EXISTS stock_bars (
    -- 数据列（时间序列数据）
    ts TIMESTAMP,                    -- K线时间戳
    open FLOAT,                      -- 开盘价
    high FLOAT,                      -- 最高价
    low FLOAT,                       -- 最低价
    close FLOAT,                     -- 收盘价
    volume BIGINT,                   -- 成交量（手）
    amount FLOAT,                     -- 成交额（元）
    open_interest BIGINT,              -- 持仓量（期货专用，股票为0）
    settlement FLOAT,                 -- 结算价（期货专用，股票为0）
    change FLOAT,                    -- 涨跌额
    change_rate FLOAT                 -- 涨跌幅（%）
) TAGS (
    -- 标签列（静态数据）
    symbol NCHAR(20),               -- 股票代码
    market NCHAR(10),               -- 市场
    interval NCHAR(10),              -- K线周期（1min、5min、15min、30min、60min、1day、1week、1month）
    name NCHAR(50)                 -- 股票名称
);
```

### 表8: stock_top_list（股票龙虎榜表）

**说明**: 存储股票龙虎榜数据，记录上榜原因和买卖情况

```sql
CREATE STABLE IF NOT EXISTS stock_top_list (
    -- 数据列（时间序列数据）
    ts TIMESTAMP,                    -- 上榜日期
    reason NCHAR(100),              -- 上榜原因
```
